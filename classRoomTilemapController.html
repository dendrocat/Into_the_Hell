Detailed Description

Контроллер для управления тайлмапами комнаты. Реализует интерфейс IRoomTilemapController. Этот компонент должен быть удалён с игрового объекта после генерации уровня через Destroy. Также уничтожает связанный экземпляр IDoorTilemapController.
