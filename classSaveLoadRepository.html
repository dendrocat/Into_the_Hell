Подробное описание

Обрабатывает сериализацию в JSON и операции с файлами для данных игры.

Управляет созданием, удалением файлов сохранения и конвертацией JSON с использованием структуры GameData. Реализует автоматическую инициализацию новой игры при отсутствии сохранения.
