Открытые члены

void PauseGame ()
 Включает паузу: отображает UI паузы и приостанавливает игру.
 
void UnPauseGame ()
 Выключает паузу: скрывает UI паузы и возобновляет игру.
 
void ToMainMenu ()
 Переход в главное меню из паузы. Сначала снимает паузу, затем вызывает переход.
 
void Activate ()
 Активирует подписки на события ввода для управления паузой. Отписывает от события нажатия паузы и подписывается на отмену.
 
void Deactivate ()
 Деактивирует подписки на события ввода для управления паузой. Подписывается на событие нажатия паузы и отписывается от отмены.
 

Подробное описание

Контроллер паузы игры. Управляет отображением паузы, приостановкой и возобновлением игры, а также переходом в главное меню из паузы.
