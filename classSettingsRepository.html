Подробное описание

Обеспечивает типобезопасное хранение настроек в PlayerPrefs с автоматическим добавлением префиксов ключей.

Методы

◆ Load()

static Dictionary< string, object > SettingsRepository.Load ( IEnumerable< string >  keys)
inlinestatic

Загружает настройки из PlayerPrefs по списку ключей.

Аргументы
keysКоллекция имён настроек для загрузки.
Возвращает
Словарь загруженных значений с исходными ключами.

Автоматически:

  • Определяет тип значения по префиксу ключа.
  • Конвертирует сохранённые данные в исходные типы.
  • Пропускает отсутствующие ключи без ошибок.

◆ Save()

static void SettingsRepository.Save ( IDictionary< string, object >  data)
inlinestatic

Сохраняет коллекцию настроек в PlayerPrefs.

Аргументы
dataПары "ключ-значение" для сохранения, где ключ - имя настройки, значение - её значение.

Автоматически:

  • Добавляет префиксы для предотвращения конфликтов ключей.
  • Конвертирует значения в поддерживаемые типы.
  • Логирует ошибки для неподдерживаемых типов.

Отображает поддерживаемые типы значений в префиксы ключей PlayerPrefs.

Используется для различения типов значений при сохранении и загрузке. Поддерживаются типы:

  • int
  • float
  • bool
  • string
