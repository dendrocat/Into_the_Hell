Подробное описание

Компонент NPC, с которым можно взаимодействовать для запуска диалога.

Требует компонент Collider2D для обнаружения игрока в зоне взаимодействия. При входе игрока в триггер отображается подсказка, при выходе - скрывается. При взаимодействии запускает диалог с использованием DialogManager.
